## Parts

- Pi Pico H
- 2 Joysticks
- PCF8591 (ADC mux - I2C)
- HC-08 (bluetooth)
- TP4056 (USB-C Lipo charger)
- 18650 Lipo 3.7

## Data Protocol
- Left button
  - Lx: M/L/R\n
  - Ly: M/U/D\n
  - Lb: 1/0\n (not implemented yet)
- Right button
  - Rx: M/L/R\n
  - Ry: M/U/D\n
  - Rb: 0/1\n (not implemented yet)
